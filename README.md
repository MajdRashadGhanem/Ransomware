# نظام محاكاة تشفير الملفات (Educational Ransomware Simulation)

هذا المشروع عبارة عن نموذج تعليمي مكتوب بلغة **Python** يوضح آليات تشفير الملفات باستخدام خوارزمية **AES-128** في نمط **CTR**. الهدف من المشروع هو فهم كيفية تعامل البرمجيات مع أنظمة الملفات وطرق التشفير المتماثل.

## ⚠️ تحذير هام (Disclaimer)
هذا البرمجيات مخصصة **للأغراض التعليمية والأمنية فقط**. لا تقم بتشغيل هذا الكود على جهازك الشخصي أو على أي بيانات هامة. المطور غير مسؤول عن أي سوء استخدام أو فقدان غير مقصود للبيانات.

---

## 🚀 الميزات
*   **تشفير الملفات:** يقوم بالبحث عن ملفات محددة (صور، مستندات، قواعد بيانات) وتشفيرها داخل مسار محدد.
*   **واجهة مستخدم (GUI):** واجهة بسيطة باستخدام `Tkinter` تطلب من المستخدم إدخال مفتاح فك التشفير.
*   **خوارزمية قوية:** يستخدم مكتبة `PyCryptodome` لتطبيق تشفير AES.
*   **دعم الامتدادات:** يستهدف قائمة واسعة من امتدادات الملفات الشائعة.

## 🛠️ المتطلبات
*   Python 3.x
*   مكتبة `pycryptodome`
